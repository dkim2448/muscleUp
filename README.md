# muscleUp

a workout tracking app for android that helps you log exercises, sets, and track progression over time.

## features

- **week view**: browse all weeks and select which week to log workouts
- **day view**: see all exercises and sets for a specific day
- **add workouts**: log exercises with multiple sets (weight, reps, and set number)

## tech stack

- **language**: java
- **database**: room
- **ui**: android studio, view binding

## project structure

- mainactivity: week selection screen
- daylistactivity: view days in selected week
- daydetailactivity: log and view workouts
- room database entities and daos for data persistence
